Detailed Description

Class representing a single HTTP session.

Servers requests acquired from the client using RequestHandler class to parse, interpret request and send back a response.

◆ run()

void HttpSession::run ( )

Run the session.

Note
run() method creates another shared_ptr on the HttpSession (first was created by Listener::__on_accept()) which, in turn, is again removed at the end of the scope.

At this time it's __onRead() method's responsibility to prolong object's lifetime. Method's call decides if object's should be maintained or deleted.
