Detailed Description

Main model of the application. Top layer of the application state interface.

Model class represents a single instance of the application that can interact with a user. It manages states of the application and legal transitions between them. It contains modules responsible for particular functionalities like:

  • simulation creation
  • simulation processing
  • saved simulation loading
  • adjusting app's settings
