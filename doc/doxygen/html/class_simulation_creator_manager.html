Detailed Description

Class responsible for app's behaviour in the simulation creator.

Class holds all parameters required by the creator in a shape of parameters tress. It facilitates methods responsible for serializing and deserializing parameters (within JSON format) tress as well as methods performing loading and saving parameters to the database tables assiociated with a client.

Note
SimulationCreatorManager is the only one object that simulation can be started from in a valid way.
