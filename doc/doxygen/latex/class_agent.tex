\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_agent}{Agent}} class represents a single \mbox{\hyperlink{class_agent}{Agent}} (a lorry) responsible for delivering transports. 

Class incorporates basic informations about an agent, involving machine\textquotesingle{}s and driver\textquotesingle{}s limits, identificators, actual transport specification or technical condition of the machine.
