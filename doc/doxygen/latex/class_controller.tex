\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_controller}{Controller}} class is an interface between \mbox{\hyperlink{class_request_handler}{Request\+Handler}} and \mbox{\hyperlink{class_model}{Model}} instance. 

\mbox{\hyperlink{class_request_handler}{Request\+Handler}} parses H\+T\+TP P\+O\+ST requests (that client\textquotesingle{}s action are sent with by the local Java\+Script scripts) and passes parsed action to the \mbox{\hyperlink{class_controller}{Controller}}. \mbox{\hyperlink{class_controller}{Controller}}\textquotesingle{}s aim is to interpret this action and call appropriate methods on the \mbox{\hyperlink{class_model}{Model}}. ~\newline
