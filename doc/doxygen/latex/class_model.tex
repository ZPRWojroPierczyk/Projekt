\doxysubsection{Detailed Description}
Main model of the application. Top layer of the application state interface. 

\mbox{\hyperlink{class_model}{Model}} class represents a single instance of the application that can interact with a user. It manages states of the application and legal transitions between them. It contains modules responsible for particular functionalities like\+:


\begin{DoxyItemize}
\item simulation creation
\item simulation processing
\item saved simulation loading
\item adjusting app\textquotesingle{}s settings 
\end{DoxyItemize}
