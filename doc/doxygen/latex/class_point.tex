\doxysubsection{Detailed Description}
Class represents a single point on the road. 

Accumulates all essential informations about the route point as velocity limits, accidents and traffics probabilities. As point coordinates are represented by boost\+::numeric\+::ublas\+::vector instance get\+Vector() method makes it able to perform algebralical operations on the points calculating e.\+g. their distance or direction pointed by two Points
