\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_route}{Route}} class represents an agent\textquotesingle{}s route computed by the planner. 

This class is a basic route containing only basic informations about route involving set of points constituting aproximate shape of the road and summary lenth. Class shares simple interface to get basic informations about the route.
