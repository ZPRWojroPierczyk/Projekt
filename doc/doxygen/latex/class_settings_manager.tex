\doxysubsection{Detailed Description}
Class responsible for keeping track of application instance\textquotesingle{}s global settings. 

Global settings involve graphic and auto settings. Class facilitates interface to load and save client\textquotesingle{}s settings to the database when the client is logginig in and logging out.
