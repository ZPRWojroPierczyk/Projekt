\doxysubsection{Detailed Description}
Main program\textquotesingle{}s class representing core of the simulation. 

This class holds informations about simulation\textquotesingle{}s core involving agents\textquotesingle{} and transports\textquotesingle{} state. The class shares an interface that facilitates simulation\textquotesingle{}s flow controll.
